NutraGive is a community-driven food assistance platform that connects donors with individuals and families experiencing food insecurity. Recipients fill out a short form describing their      
  situation, and an AI generates a compelling, dignified funding request on their behalf. Donors can browse open requests, filter by tag or price range, and fund them with a single payment — the 
  recipient then receives a coupon code redeemable for real groceries.                                                                                                                             
                                                                                                                                                                                                   
  Tech Stack:                                                                                                                                                                                      
  - Framework: Next.js 16 (App Router, server + client components)                                                                                                                                 
  - Frontend: React 19, Tailwind CSS v4                                                                                                                                                            
  - Backend: Supabase (PostgreSQL, Row Level Security, Auth)                                                                                                                                       
  - AI: Featherless AI (Meta Llama 3.1 8B) — request generation + support chatbot                                                                                                                  
  - Email: Brevo — transactional funding notifications                                                                                                                                            
  - Language: TypeScript
